#Introdução sobre a linguagem Go

##Por que usar Go

Go oferece desempenho excepcional com implantação em binário único, concorrência nativa, compilação rápida e uma biblioteca padrão abrangente. É uma linguagem simples, fácil de aprender e manter. Destaca-se no desenvolvimento de serviços web, microsserviços, ferramentas de linha de comando (CLI) e softwares de sistema.

##História do Go

Criada no Google em 2007 por Griesemer, Pike e Thompson. Foi anunciada publicamente em 2009, com a versão 1.0 lançada em 2012. Entre os principais marcos estão os módulos (Go 1.11) e os genéricos (Go 1.18). Foi projetada para o desenvolvimento de software em larga escala, combinando eficiência e simplicidade.

##Configurando o Ambiente

Instale o Go pelo site oficial configure o PATH e prepare o ambiente de trabalho. Configure um editor com suporte para Go (VS Code, GoLand, Vim/Emacs). Utilize módulos para gerenciamento de dependências. Verifique a instalação com go version e teste com um programa simples.
