# 🚀 Introdução à Linguagem Go

> Uma linguagem moderna, eficiente e criada para o desenvolvimento de software em larga escala.

---

# 📌 O que é Go?

Go, também conhecida como **Golang**, é uma linguagem de programação desenvolvida pelo Google com foco em:

* ✅ simplicidade
* ✅ desempenho
* ✅ concorrência
* ✅ produtividade
* ✅ escalabilidade

Ela foi projetada para facilitar o desenvolvimento de aplicações modernas e de alta performance.

---

# 💡 Por que usar Go?

Go oferece diversas vantagens que a tornaram uma das linguagens mais populares para backend e infraestrutura.

---

# ⚡ Principais vantagens

| Recurso                   | Benefício                         |
| ------------------------- | --------------------------------- |
| 🚀 Alto desempenho        | Execução rápida e eficiente       |
| ⚙️ Compilação rápida      | Builds extremamente rápidos       |
| 🔄 Concorrência nativa    | Goroutines e channels             |
| 📦 Binário único          | Implantação simplificada          |
| 🧩 Biblioteca padrão rica | Muitas funcionalidades integradas |
| 📚 Simplicidade           | Fácil de aprender e manter        |

---

# 🌐 Onde Go é utilizado?

Go se destaca principalmente no desenvolvimento de:

* ✅ APIs
* ✅ microsserviços
* ✅ servidores web
* ✅ ferramentas CLI
* ✅ softwares de sistema
* ✅ aplicações distribuídas

---

# 🏛️ História do Go

A linguagem Go foi criada no Google em 2007 por:

* Robert Griesemer
* Rob Pike
* Ken Thompson

# 🛠️ Configurando o Ambiente

Para começar a programar em Go, é necessário configurar corretamente o ambiente de desenvolvimento.

---

# 📥 Instalando Go

O Go pode ser instalado através do site oficial.

Após a instalação:

* configure o `PATH`
* prepare o ambiente de trabalho
* instale um editor compatível

---

# 💻 Editores recomendados

| Editor  | Característica            |
| ------- | ------------------------- |
| VS Code | Leve e popular            |
| GoLand  | IDE especializada         |
| Vim     | Alta produtividade        |
| Emacs   | Extremamente configurável |

---

# 📦 Gerenciamento de dependências

Go utiliza módulos para gerenciar dependências.

O sistema oficial é chamado de:

```text id="gww6dd"
Go Modules
```

---

# 🔍 Verificando instalação

Após instalar o Go, utilize:

```go id="ccn3o2"
go version
```

Esse comando verifica se a instalação foi concluída corretamente.

---

# 💻 Primeiro teste

Você também pode validar o ambiente criando um programa simples.

```go id="t6sg7p"
package main

import "fmt"

func main() {
    fmt.Println("Go funcionando corretamente!")
}
```

---

# ▶️ Executando o programa

```go id="smqby0"
go run main.go
```

> 🚀 Se a mensagem aparecer no terminal, o ambiente está pronto para desenvolvimento.

---

# ⚠️ Cuidados importantes

> ⚠️ Variáveis de ambiente configuradas incorretamente podem impedir o funcionamento do Go.

> 💡 Utilize módulos desde o início do projeto para melhor gerenciamento de dependências.
